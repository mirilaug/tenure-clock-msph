What is the Tenure Timeline Calculator?
The calculator was built using Columbia University policy documents to model the tenure-track process. It takes an appointment start date and generates a full timeline of milestones — department vote, COAP stages, TRAC review, promotion, and up-or-out date — with all dates calculated automatically. It also handles clock extensions, which can cause real confusion about which deadlines shift and which don't. For example, the fourth-year review is anchored to the hire date and is not affected by extensions.
How do I access it, and is it publicly available?
You can access the demo version at https://mirilaug.github.io/tenure-clock-msph/. In order to allow demo access it has been made publicly available; a significant disclaimer is included on the page.
Who is it for?
Right now it is mainly for faculty, but there is also a departmental view that allows administrators or mentors to enter a name and start date for multiple faculty members at once and compare key deadlines side-by-side.
Is this a final or provisional version?
This is a provisional version, subject to administrative review and policy verification. It currently applies only to research faculty on the 7-year tenure track.
Can I export the data generated?
Yes. The individual calculator provides export options for PDF, calendar (.ics), and spreadsheet (.csv). The departmental view also supports PDF and .csv export.
Privacy — what data does this tool collect or store?
This is a demo version and does not ask for a name in the individual calculator — only a start date. The departmental view does include a name field for convenience. No data is transmitted or stored anywhere; everything runs locally in your browser. As a precaution, please avoid entering real names in the departmental view.
