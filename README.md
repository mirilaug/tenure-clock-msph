# tenure-clock-msph

Draft version of a calculator for people on the tenure track at Columbia Mailman School of Public Health. Please note, currently this is not an officially approved tool — for demonstration purposes only.

## How to use

1. **Individual calculator** — Enter a faculty member's appointment start date. The full tenure timeline generates automatically. Add clock extensions (e.g. parental leave) if applicable, then export the timeline as a PDF, .ics calendar file, or .csv spreadsheet.
2. **Departmental view** — Add multiple faculty members to compare key deadlines side-by-side across the department and export a combined PDF or .csv.

> **Important:** All milestone dates are estimates based on the assumptions built into this tool and have not yet been verified against current Columbia University policy. Verify all deadlines with the Office of Faculty Affairs before sharing with faculty.
